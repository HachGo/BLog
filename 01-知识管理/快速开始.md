# 快速开始

本指南将帮助你快速上手知识管理系统。

## 第一步：安装工具

1. 下载并安装 [Obsidian](https://obsidian.md/)
2. 克隆本仓库到本地
3. 用 Obsidian 打开仓库文件夹

## 第二步：基本操作

### 创建笔记

```
Ctrl + N (Windows/Linux)
Cmd + N (Mac)
```

### 搜索笔记

```
Ctrl + O (Windows/Linux)
Cmd + O (Mac)
```

## 第三步：使用链接

### 内部链接

```markdown
[[另一篇笔记]]
```

### 嵌入内容

```markdown
[[笔记|显示文本]]
```

## 进阶技巧

- 使用 `#标签` 组织内容
- 使用 `==高亮==` 标记重要内容
- 利用 `[[^块ID]]` 引用特定段落

> **提示**: 保持笔记原子化，每篇笔记只讨论一个主题。
